# LogistiX

**LogistiX** is a next-generation Smart Transport Operations Platform built natively as an Odoo ERP application.

## Overview
LogistiX centralizes the complexities of modern transport and logistics into a single, cohesive ERP environment. 

### Core Modules
* **Vehicle & Fleet Management:** Real-time tracking of vehicle capacity, assignments, and availability.
* **Driver & HR Management:** Automated safety compliance, license tracking, and hours-of-service logging.
* **Trip & Dispatch Operations:** Seamless trip creation, driver assignment, and status transitions.
* **Expense & Fuel Tracking:** Integrated workflows for logging and approving trip-related expenses.

## Architecture
- **Backend:** Odoo ORM (Python)
- **Database:** PostgreSQL
- **Frontend:** Odoo XML Views & Dashboards
